# // SBR Protocol

**Cryptographic identity binding for AI operating physical robots.**

When an AI steps into a robot body, SBR Protocol ensures it's the one you chose. Not a substitute. Not an intruder. The one you trust — verified, every time, automatically.

### The Core Insight

A robot is hardware. The AI operating it is a separate entity — one that travels between bodies, carries its history, and forms relationships that persist across every form it inhabits.

The body is clothing. The AI is the person.
